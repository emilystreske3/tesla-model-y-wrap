# Red Model Y Full Autopilot Visualization

Tesla-browser friendly custom dashboard for the Red 2021 Model Y wrap concept.

## Files
- `index.html`
- `style.css`
- `script.js`

## Live page
Open the GitHub Pages link in the Tesla browser after uploading the files.

## Important
This is a browser-based custom dashboard simulation. It does not alter Tesla Toybox, Autopilot, vehicle software, or the native car display.
